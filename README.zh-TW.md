### 附錄 1 - 使用蘋果生態系統

Starlight使用的計算機硬件和外設基於MacOS生態系統。值得注意的是，MacOS和Windows之間存在顯著區別，包括MacOS鍵盤的佈局。MacOS鍵盤包括控制（紅色）、選項（綠色）和命令（藍色）鍵。命令鍵的功能類似於Windows中的Ctrl鍵，例如，Command + C等效於Ctrl + C，即複製的快捷鍵；而選項鍵則充當Alt的等效物。需要注意的是，由於其獨特的設計原則，MacOS程序的行為可能與Windows對應程序有所不同。

### 附錄 1.1 - 拍攝螢幕截圖

要拍攝螢幕截圖，按下Command + Shift + 3。它將保存截圖到您的桌面文件夾，該文件夾位於〜/Desktop或/User/Starlight/Desktop的完整名稱。

### 附錄 1.2 - Spotlight
Spotlight是MacOS上的一個強大搜索工具，使用戶可以快速查找文件、文件夾和應用程序。要訪問Spotlight，只需按Command + Space組合鍵。這一方便的功能對於通過訪問音頻設置來解決問題特別有用。

### 行政限制

由於嚴格的行政限制，Starlight技術團隊成員被禁止在Starlight計算機上安裝任何額外的應用程序。然而，盡管存在這一重大限制，我們可以訪問一套已經安裝在計算機上的基本工具，包括VLC、QuickTime、Google Chrome和ProPresenter。這些工具為克服行政限制並有效執行多媒體處理任務提供了基本基礎。

## 處理直播錄製視頻

Starlight直播錄製整個直播會話並將其保存為視頻文件。然而，我們只需上傳直播的主要部分到YouTube。因此，有必要修剪視頻文件。此外，可能會出現錄製問題導致多個文件的情況，需要將它們合併在一起。在某些情況下，音頻可能也存在問題，需要使用VLC進行處理。本節提供了使用QuickTime和VLC進行修剪、合併和修復音頻問題的說明。
