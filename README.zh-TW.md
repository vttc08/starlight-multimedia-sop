### 附錄 1 - 使用蘋果生態系統

Starlight使用的計算機硬件和外設基於MacOS生態系統。值得注意的是，MacOS和Windows之間存在重大區別，包括MacOS鍵盤的佈局。MacOS鍵盤包括控制（紅色）、選項（綠色）和命令（藍色）鍵。命令鍵的功能類似於Windows中的Ctrl鍵，例如，Command + C等同於Ctrl + C，即複製的快捷鍵；而選項鍵則相當於Alt。重要的是要認識到，由於其獨特的設計原則，MacOS程序的行為可能與Windows對應程序有所不同。

![](assets/2024-01-11-22-46-35.png)

### 附錄 1.1 - 截圖

要截圖，按Command + 3。它將保存截圖在您的桌面文件夾中，位於〜/Desktop或完整名稱的/User/Starlight/Desktop。

### 附錄 1.2 - Spotlight

Spotlight是MacOS上的一個強大的搜索工具，它使用戶能夠快速定位文件、文件夾和應用程序。要訪問Spotlight，只需按Command + Space鍵組合。這一便利功能對於通過訪問音頻設置來解決音頻問題尤其有用。

### 行政限制

由於嚴格的行政限制，Starlight技術團隊成員被禁止在Starlight計算機上安裝任何其他應用程序。然而，儘管存在這一重大限制，我們可以訪問已經安裝在計算機上的一套基本工具，包括VLC、QuickTime、Google Chrome和ProPresenter。這些工具提供了基本的基礎，以克服行政限制並有效執行多媒體處理任務。
