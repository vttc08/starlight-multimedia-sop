### 附錄 1 - 使用蘋果生態系統

Starlight使用的電腦硬體和周邊設備基於MacOS生態系統。值得注意的是，MacOS和Windows之間存在顯著區別，包括MacOS鍵盤的布局。MacOS鍵盤包含控制（紅色）、選項（綠色）和命令（藍色）鍵。命令鍵的功能類似於Windows中的Ctrl鍵，例如，Command + C等同於Ctrl + C，即複製的快捷鍵；而選項鍵則相當於Alt。需要注意的是，由於其獨特的設計原則，MacOS程序的行為可能與Windows相異。

### 附錄 1.1 - 截圖

要截圖，請按Command + Shift + 3。它將保存截圖到桌面文件夾，位於〜/Desktop或完整名稱/User/Starlight/Desktop。

### 附錄 1.2 - Spotlight
Spotlight是MacOS上一個強大的搜索工具，使用戶能夠快速定位文件、文件夾和應用程序。要訪問Spotlight，只需按Command + Space鍵組合。這一方便的功能對於通過訪問音頻設置來排除故障很有用。 

### 行政限制

由於嚴格的行政限制，Starlight技術團隊成員被禁止在Starlight電腦上安裝任何附加應用程序。然而，儘管受到這一重要限制，我們可以訪問一組已經安裝在計算機上的基本工具，包括VLC、QuickTime、Google Chrome和ProPresenter。這些工具提供了一個基本的基礎，以克服行政限制並有效執行多媒體處理任務。

## 處理直播錄影

Starlight直播錄影整個直播會話並將其保存為視頻文件。然而，我們只需要上傳主要部分到YouTube。因此，有必要修剪視頻文件。此外，有時錄製問題會導致多個文件，需要將它們合併在一起。在某些情況下，音頻也可能存在問題，需要使用VLC進行處理。本部分提供了使用QuickTime和VLC進行修剪、合併和修復音頻問題的說明。

### 在QuickTime中合併

在存在錄製問題且有多個視頻文件的情況下，需要將它們合併在一起。要合併它們，在QuickTime中打開並單擊編輯 > 添加片段到末尾。然後，選擇要合併的所有視頻文件。完成後，單擊文件 > 另存為，並為視頻命名。
